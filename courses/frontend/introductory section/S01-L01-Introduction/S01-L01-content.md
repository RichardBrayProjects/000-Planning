### b) Transcript

Hello and welcome to the course.

If you're interested in learning modern frontend web development, you're in exactly the right place.

In this course we're going to build a deep understanding of the technologies that power modern web interfaces. We'll start with the fundamentals of HTML, CSS and JavaScript, move into TypeScript so that we can write safer and more scalable code, and then build modern applications using React.

But this course is not just about learning theory.

Throughout the course we'll build a complete real-world frontend application step by step, using the same tools and development workflow used by professional developers today.

My goal is not simply to show you how to write code, but to help you understand how modern frontend development actually works — how applications are structured, how components interact, and to help you gain a solid understanding of the development tools and technologies used in professional environments.

By the end of this course you'll be able to build modern frontend applications confidently, and established a firm foundation for learning and growing as a developer.
