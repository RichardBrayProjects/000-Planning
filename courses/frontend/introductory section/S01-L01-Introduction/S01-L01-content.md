# Slide 1

**Title**

Welcome to the Course

**Bullet Points**

• Modern Frontend Web Development
• HTML, CSS, JavaScript, TypeScript, React
• Build a complete real-world application

---

### Narrative

Hello and welcome to the course.

In this course you'll learn modern frontend web development using the core technologies used by professional developers today.

We'll start with the fundamentals of HTML, CSS, and JavaScript, move into TypeScript, and then build modern user interfaces using React.

Throughout the course we'll apply everything we learn by building a complete real-world frontend application step by step.

Getting Started

**Bullet Points**

• Clone the course project

---

### Narrative

Before we begin writing code, we'll first make sure everything is ready so you can follow the course smoothly.

We'll start by looking at the course repositories so you understand how the materials and project are organized.

Then we'll set up your development environment, including Visual Studio Code, and make sure your machine is ready for modern frontend development.

Once everything is configured, you'll clone the course project and run the final version of the application so you can see what we'll be building throughout the course.
