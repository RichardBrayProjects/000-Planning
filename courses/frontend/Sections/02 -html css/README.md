# HTML and CSS course

This course keeps each lesson small and focused.

Important design choices:
- lesson 02 has no code changes
- the course does not always grow continuously; code can be removed when that makes the teaching clearer
- the last lessons build toward the final required source files

Final lesson sequence:
- 14 HTML document structure
- 15 Scrolling
- 16 Flexbox
- 17 Flexbox with scrolling
- 18 Hover and final UI

Every lesson contains:
- `README.md` — a short lesson narrative
- `changes.md` — readable edits from the previous lesson
- code files for that lesson
