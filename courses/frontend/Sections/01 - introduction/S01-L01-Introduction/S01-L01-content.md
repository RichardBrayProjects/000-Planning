### Narrative

Hello and welcome to this course on modern frontend web development.

We're going to start with a few short tutorials on HTML, CSS, and JavaScript.

We will then progress onto React and Typescript, where we'll be building a photo-sharing website frontend

This way you don't just learn a whole bunch of React programming techniques,

but you'll also see how they're used in a practical real-world situation.

# Slide 2

**Title**

Getting Started

**Bullet Points**

• Navigate the course repositories
• Set up your machine and development tools
• Clone the course project
• Run the final version of the application

---

### Narrative

Before we begin writing code, we'll first make sure everything is ready so you can follow the course smoothly.

We'll start by looking at the course repositories so you understand how the materials and project are organized.

Then we'll set up your development environment, including Visual Studio Code, and make sure your machine is ready for modern frontend development.

Once everything is configured, you'll clone the course project and run the final version of the application so you can see what we'll be building throughout the course.

We'll then take a short walk through the application so you can understand the project we'll be building throughout the course.

Finally, I'll share a few simple tips that will help you follow the course efficiently and get the most value from each lesson.

After that, we'll be ready to begin learning the core fundamentals of modern frontend development.

I'll see you in the next lesson.
