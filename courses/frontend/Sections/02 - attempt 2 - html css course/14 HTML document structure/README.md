# Lesson 14 — HTML document structure

In this lesson, you introduce the full HTML document built from Emmet with `!`.

You teach:
- `<!doctype html>`
- `html`, `head`, and `body`
- semantic tags: `header`, `nav`, `main`, and `footer`
- linking `index.css`

This resets the course onto the final project structure that the last lessons will build on.
