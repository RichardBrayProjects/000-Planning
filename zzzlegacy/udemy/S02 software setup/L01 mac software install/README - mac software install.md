# Install Homebrew

- in a browser, go to brew.sh and copy the installation command provided by homebrew

- paste it into a terminal window and run it

- NOTE : it will finish by asking you to run (3) commands to setup the required homebrew environment variables 

- RUN THESE COMMANDS in your terminal !

# Softare installation

- edit the setup.sh bash script provided with this lesson 

- remove any applications which you already have installed

- open a terminal window where the setup.sh script is located

- make the script executable and execute it

```zsh
chmod u+x ./setup.sh
./setup.sh
```
