
# install deno

## deno - mac installation instructions

```bash
brew install deno
```
## deno - windows installation instructions

Open an admin terminal/powershell  

```bash
choco install deno
```

# install the deno Jupyter kernel

```bash
 deno jupyter --install
```

# install the Microsoft Jupyter extension into VS Code

```bash
 code --install-extension ms-toolsai.jupyter
```

# Install the Deno extension into VS Code

```bash
 code --install-extension denoland.vscode-deno
```

# Open or Create a Jupyter Notebook

to create a new one open the VS Code command palette with

```
Create: New Jupyter Notebook
```

# Set the kernel

on the top far right select the kernel

# To Uninstall Deno :

Uninstall Deno Jupyter kernel:

```bash
deno jupyter --uninstall
```

Uninstall Deno:

```bash
brew uninstall deno
```
