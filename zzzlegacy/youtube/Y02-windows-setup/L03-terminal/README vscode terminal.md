# Set the default terminal

- select the small dropdown on the far right of the terminal header
- choose Select Default Profile
- select gitbash as the default profile

# Configure gitbash prompt

- open a terminal
- run the command

```bash
nano .bash_profile
```

- add the following to the file

```bash
PS1="$ "
```

- press Control-O to write the file (and enter)
- press Control-X to exit

- restart the terminal
